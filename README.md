# 2020 summer plan

### Git使用方式

#### 提交

~~~
git add [files] / git add .
git commit -m "commit message"
git push -u origin summer
~~~

#### 切换分支

~~~
git checkout summer|master|others
~~~

#### 创建本地分支

远程分支已经创建好了，创建本地分支后push即可

~~~
git branch summer|master|others
~~~

#### 显示所有分支

~~~
git branch
~~~
